Context Commander

Customize Windows Explorer context menus without downloading an executable.
Examples
  • Run file with Node.js
  • Start Web server here
How it works

Explorer context menu entries are configured in the registry. Instead of configuring your registry manually, this page will provide you with reusable, redistributable .reg files for installing and uninstalling your context menu entry.

Why no executable?

Maybe you don't want to download a program just to run it once. Maybe you can't use 3rd party programs on your work PC. Or maybe, like me, you just want to see exactly what changes are being made to your registry.
